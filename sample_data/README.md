# Sample STAC Dataset

This directory contains a ready-to-ingest example dataset for demonstrating customer-side STAC ingestion and management.

## Contents

* One STAC Collection
* Three STAC Items belonging to that collection
* Metadata-only examples (no data assets required)

## Purpose

The sample dataset allows you to:

* Create a new Collection
* Ingest multiple Items
* Modify existing STAC entities
* Delete Items or the Collection
